# 토스 AI 매크로 트레이더 for macOS

토스증권 API와 코덱스, 클로드, 제미나이 CLI를 결합해 조건 기반 자동매매와 AI 전략 분석을 지원하는 macOS 네이티브 주식 매크로 앱입니다.

## 핵심 구성

- SwiftUI 기반 macOS 네이티브 앱
- 토스증권 Open API 기반 시세, 계좌, 보유 종목, 주문 요청 준비
- Swift Charts 기반 대시보드
- macOS 키체인을 통한 토스 API 인증 정보 저장
- 관심 종목, 전략, AI 명령어 설정, 안전 설정, 주문 로그의 로컬 저장
- 전략 조건 충족 알림을 위한 UserNotifications 연동
- 코덱스, 클로드, 제미나이 CLI 실행 브리지

## 안전 정책

- AI 출력은 참고용 분석입니다.
- 라이브 주문은 기본적으로 잠겨 있습니다.
- 전략 조건이 충족되어도 라이브 주문 관문을 명시적으로 열기 전에는 알림과 로그만 생성합니다.
- 시장가 주문은 기본적으로 추가 확인을 요구합니다.
- API 또는 네트워크 오류가 발생하면 자동매매를 중지할 수 있습니다.
- 중복 주문 방지와 쿨다운 검사는 전략 조건 처리 경로에 포함됩니다.
- 모든 조건 충족 판단은 전략, 모드, AI 검토 상태, 주문 드래프트, 결과와 함께 기록됩니다.

## 단계별 계획

### 1단계

- 토스 API 키 등록
- 계좌 조회
- 보유 종목 조회
- 현재가 조회
- 관심 종목 관리
- 가격 조건 알림
- AI CLI 연결 테스트

### 2단계

- 전략 조건 저장
- 반자동 주문 흐름
- 손절/익절 알림
- 코덱스 전략 조건 생성
- 클로드 리스크 분석
- 제미나이 시장 요약
- 주문 로그 저장

### 3단계

- 완전 자동 주문 안전 관문
- WebSocket 실시간 감시
- 주문 전 AI 검토
- 백테스트
- 일일 매매 리포트
- 미국장 야간 감시

## 고지

본 앱은 투자 수익을 보장하지 않습니다. 모든 투자 판단과 손실 책임은 사용자에게 있습니다. AI 분석은 참고용이며 매수 또는 매도 결정을 보장하지 않습니다. 토스증권 API 제공 범위, 주문 지원 여부, 호출 제한, 약관은 공식 토스증권 문서를 따릅니다.
