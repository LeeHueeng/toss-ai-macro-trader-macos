## 이번 적용 방향

- 쉬운 매크로 화면에 최근 차트 기준 간단 백테스트 미리보기 추가
- 실전 자동주문보다 모의/승인 흐름을 기본값으로 유지
- 초보자용 문장으로 위험 안내를 강화
- 전략을 복잡한 내부 조건 대신 `신호`, `예상 손익`, `가장 나빴던 구간`으로 먼저 보여준다
